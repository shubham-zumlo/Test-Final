# 📓 My Journal

A simple, beautiful journaling app that runs entirely in your browser — no installation needed!

## Preview

![My Journal App](https://github.com/user-attachments/assets/f8ea9045-fa8f-47a0-ad4a-f2839fdecc34)

## How to Use (No coding required!)

1. **Download** — Click the green **Code** button at the top of this page → **Download ZIP** → Unzip the file on your computer
2. **Open** — Go into the unzipped folder and find the file called **`index.html`**
3. **Launch** — Double-click `index.html` — it opens in your browser and is ready to use!

## Features

- ✏️ Write journal entries in the text box
- 💾 Save entries with the **Save Entry** button
- 📋 View all your previous entries below
- 🗑️ Delete any entry with the ✕ button
- 💾 Entries are saved in your browser automatically (no account needed)
